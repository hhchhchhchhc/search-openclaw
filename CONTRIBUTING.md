# Contributing

欢迎补充和修正这个仓库里的内容。

优先欢迎以下类型的贡献：

- 搜索 API 选型对比
- 免费额度和计费变动
- OpenClaw 中的配置经验
- 多搜索源组合方案
- 实际踩坑和规避方法

## 提交建议

请尽量遵循下面几条：

- 新增内容优先放到 `docs/`
- 结论尽量写清楚适用场景，而不是只列名词
- 涉及免费额度、价格、配额时，注明“可能变化”
- 如果引用社区经验，尽量写成“社区常见说法”，避免写成官方承诺
- 优先补充对 Agent 友好的判断，而不是泛泛而谈搜索引擎本身

## 风格

这个仓库的文档风格偏实用：

- 先给结论
- 再讲适合谁
- 最后讲取舍和注意事项

尽量减少空话，优先帮助读者更快做选型。
